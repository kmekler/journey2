Journey README 12/1/14

The purpose of this application is to allow users to keep track of their favorite running spots while also recording how they felt during each individual run using our journal feature. 

* Ruby version: 2.1.2

* System dependencies include PosgreSQL, bcrypt, rSPEC and Bootstrap-sass

* Configuration

* Database created via PostgreSQL

* Deployed with Heroku: https://journey3.herokuapp.com/
